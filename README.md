# Kastus Tetris

A modern, beautiful implementation of the classic Tetris game.

## Features

- Classic Tetris gameplay with all 7 tetrominos
- SRS (Super Rotation System) with wall kicks
- Ghost piece showing landing position
- Hold piece functionality
- Next 3 pieces preview
- Progressive difficulty levels
- Beautiful visual effects and animations

## How to Play

### Desktop Controls

- **←→** - Move piece left/right
- **↑** - Rotate clockwise
- **Z** - Rotate counter-clockwise
- **↓** - Soft drop (earn 1 point per cell)
- **Space** - Hard drop (earn 2 points per cell)
- **C** - Hold current piece
- **P** - Pause game

### Mobile Controls

When playing on a mobile device, you'll see touch-friendly buttons:

- **←→↓** - Direction buttons for movement
- **↻** - Rotate button
- **DROP** - Hard drop button
- **HOLD** - Hold piece button
- **⏸** - Pause button

**Swipe Gestures** (on game board):
- Swipe left/right - Move piece
- Swipe down - Soft drop
- Swipe up - Rotate piece

**Haptic Feedback**: Feel subtle vibrations when tapping buttons (on supported devices)
